Alias values may take one of 3 forms:
1) "A" - the single number A. Example: "12" <=> { 12 }
2) "A-B" - any number inbetween A and B (inclusive). Example: "12-15" <=> { 12, 13, 14, 15 }
3) "An+B" - any number that can be made by setting n equal to a positive integer. Example: "2n+1" <=> { 1, 3, 5, 7, 9, ... }

You can also combine alias values with "|". Example: "2-4|15" <=> { 2, 3, 4, 15 }

Alias names can contain letters, numbers, or underscores, but must start with a letter. Example: "EVEN_COL_2"


{ "BLACK": 2 } means if you bet $1 on the "BLACK" alias and it hits, then you get back $2.


{ "RED": 1 } means bet $1 on the "RED" alias to start.


{ "ODD": "$EVEN = $EVEN * 2" } means anytime the "ODD" alias hits, double our bet on the "EVEN" alias.

You may use addition (+), subtraction (-), multiplication (*), division (/), and parentheses in strategy values.

You may also use alias names as long as they are prefixed with a "$" and have an initial bet defined.

You may also combine strategy values with semicolons, in order to update multiple bets when one alias hits.
